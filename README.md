## Set Up
### Steps
1. (Optional) Install NVM (`https://github.com/nvm-sh/nvm`)
1. Install nodejs 16.16.0 using NVM or direct download (`https://nodejs.org/en/blog/release/v16.16.0/`)
4. Add .env file (refer to example file)
5. Run `npm install`
6. Run `npm start`
